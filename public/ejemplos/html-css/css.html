CSS (Hojas de Estilo en Cascada)

HTML - CSS

¿Qué es CSS?

CSS, que son las siglas en inglés de Cascading Style Sheets (hojas de estilo en cascada), es un lenguaje de diseño gráfico para definir y crear la presentación de un documento estructurado escrito en un lenguaje de marcado. Es muy usado para establecer el diseño visual de los documentos web, junto con HTML.

Separación de contenido y presentación

La idea principal detrás del desarrollo de CSS es separar la estructura de un documento (su contenido) de su presentación (colores, fuentes, diseño). Esto permite mejorar la accesibilidad, reducir la complejidad y repetición del código, y hacer que múltiples páginas web compartan el mismo formato al enlazar un único archivo de estilo.

Evolución del Estándar

Al igual que HTML, CSS es gestionado por el W3C. La especificación actual es CSS3, la cual, a diferencia de sus versiones anteriores, está dividida en múltiples módulos independientes. Descubre más sobre su sintaxis e historia en el artículo completo de Wikipedia sobre CSS.
