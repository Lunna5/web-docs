HTML (Lenguaje de Marcado de Hipertexto)

HTML - CSS

¿Qué es HTML?

HTML, acrónimo en inglés de HyperText Markup Language («lenguaje de marcado de hipertexto»), hace referencia al lenguaje de marcado utilizado en la creación de páginas web. Este estándar define una estructura básica y un código para la presentación de contenido de una página web, que incluye texto, imágenes, videos, entre otros elementos.

Etiquetas HTML básicas

Las etiquetas HTML son códigos utilizados para estructurar y dar formato al contenido en páginas web. Funcionan como instrucciones que indican al navegador cómo mostrar diferentes elementos. Generalmente, un elemento se compone de una etiqueta de apertura y una etiqueta de cierre correspondiente.

Gestión del Estándar

Este estándar es gestionado por el World Wide Web Consortium (W3C), una organización dedicada a la estandarización de la mayoría de las tecnologías asociadas a la web. Puedes leer más sobre su historia y características en el artículo completo de Wikipedia sobre HTML.
