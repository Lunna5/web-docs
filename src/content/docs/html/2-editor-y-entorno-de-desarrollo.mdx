<Aside title='Nota'>
Puedes configurar tu entorno de desarrollo de muchas formas diferentes y con muchas herramientas diferentes, lo importante es que encuentres una configuración que te funcione y con la que te sientas cómodo o cómoda.

Para que todos los estudiantes tengan una experiencia similar, vamos a utilizar Visual Studio Code como editor de código, y Live Server como extensión para poder visualizar nuestras páginas web en el editor de forma rápida y sencilla, aunque puedes utilizar cualquier editor de código que te guste, y cualquier extensión o herramienta que te ayude a visualizar tus páginas web en el navegador.
</Aside>

## Instalación de Visual Studio Code
Visual Studio Code es un editor de código gratuito y de código abierto desarrollado por Microsoft, que se ha convertido en uno de los editores de código más populares entre los desarrolladores web. Visual Studio Code es un editor de código ligero, rápido, y altamente personalizable, que cuenta con una gran cantidad de extensiones y herramientas para facilitar el desarrollo web.

Puedes descargar Visual Studio Code desde su sitio web oficial, y está disponible para Windows, macOS, y Linux.

:::tip[Si estas en educaandOs]
Tienes que descargar Visual Studio Code desde el centro de software de educaandOs, se encuentra en la sección de "herramientas para desarrolladores".
:::

## Instalación de Live Server
Live Server es una extensión para Visual Studio Code que permite visualizar tus páginas web en un navegador embebido dentro del editor, lo que facilita el proceso de desarrollo web al permitirte ver los cambios en tiempo real sin tener que actualizar manualmente el navegador.

Puedes instalar Live Server desde la sección de extensiones de Visual Studio Code, buscando "Live Server" e instalando la extensión desarrollada por Microsoft.
